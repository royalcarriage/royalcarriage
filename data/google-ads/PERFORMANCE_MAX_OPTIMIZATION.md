## Step-by-Step Setup in Google Ads

### Creating a New Performance Max Campaign

1. Go to **Campaigns** → **+ New Campaign**
2. Select goal: **Leads** or **Sales**
3. Select campaign type: **Performance Max**
4. Name your campaign (e.g., "PMax - Airport")
5. Set daily budget
6. Set location targeting (see above)
7. Set language: English

### Adding Asset Groups

1. In campaign, go to **Asset Groups**
2. Click **+ New Asset Group**
3. Name it (e.g., "Airport Transportation - Chicago")
4. Add **Final URL** (website)
5. Add **Headlines** (3-15, aim for 15)
6. Add **Long Headlines** (1-5, aim for 5)
7. Add **Descriptions** (2-4, aim for 4)
8. Add **Images** (see requirements below)
9. Add **Logo**
10. Set **Audience Signals**
11. Save

### Image Requirements

| Type | Ratio | Min Size | Recommended |
|------|-------|----------|-------------|
| Landscape | 1.91:1 | 600x314 | 1200x628 |
| Square | 1:1 | 300x300 | 1200x1200 |
| Portrait | 4:5 | 480x600 | 960x1200 |
| Logo (square) | 1:1 | 128x128 | 1200x1200 |
| Logo (landscape) | 4:1 | 512x128 | 1200x300 |

**Image Tips:**
- Use high-quality photos of your vehicles
- Show professional chauffeurs
- Include Chicago landmarks (for local relevance)
- No text overlay on images
- Avoid stock photos when possible

## Performance Optimization Tips

### Week 1-2: Learning Phase
- Don't make changes during learning
- Let Google optimize delivery
- Monitor impressions and clicks

### Week 3+: Optimization
1. **Check Asset Performance**
   - Go to Asset Groups → Assets
   - Replace "Low" performing assets
   - Add more "Best" performing variations

2. **Review Search Terms**
   - Insights → Search Terms
   - Add negative keywords for irrelevant searches

3. **Adjust Audience Signals**
   - Add more specific signals based on converters
   - Exclude poor-performing demographics

4. **Optimize by Time/Day**
   - Check when conversions happen
   - Adjust ad schedule if needed

### Key Metrics to Watch

| Metric | Good | Action if Poor |
|--------|------|----------------|
| CTR | >2% | Improve headlines |
| Conv. Rate | >3% | Check landing page |
| CPA | <$50 | Tighten targeting |
| ROAS | >3x | Scale budget |
